# Smart Attendance System using Face Recognition

## 📌 Overview
This project automates attendance using face recognition. It detects and recognizes faces in real-time and logs attendance.

## 🚀 Features
- Real-time face detection
- Face recognition using LBPH algorithm
- Automatic attendance logging

## 🛠️ Tech Stack
- Python
- OpenCV
- NumPy
- Pandas

## ⚙️ Setup Instructions

1. Install dependencies:
pip install opencv-python numpy pandas pillow

2. Run dataset collection:
python face_dataset.py

3. Train model:
python train_model.py

4. Start attendance system:
python recognize.py

## 📊 Output
- Attendance saved in attendance.csv
